Helpers
======================================

Additional helper functions ("utils")

.. function:: setNumThreads(numThreads)

   Sets the number of threads to use when processing

   :param numThreads: (int32) number of threads to use

.. function:: getNumThreads()

   Gets the number of threads being used for processing

   :rtype: (int32) number of threads

.. function:: setGlobalState(state)

   Sets the global random number generator state

   :param state: (uint32) state to set to

.. function:: getGlobalState()

   Gets the global random number generator state

   :rtype: (uint32) state
